X4GOOD - BANCO DE DADOS EM GRAFOS (NEO4J)
Pasta de Entrega do Trabalho Pratico

Esta pasta contem todos os entregaveis solicitados para o trabalho pratico:

1. Script de Criacao e Povoamento:
   - Arquivo: criacao_e_povoamento.cypher
   - Descricao: Script contendo os comandos Cypher para limpar a base de dados, estruturar as tabelas/nos e inserir os dados simulados completos.

2. Diagrama do Grafo:
   - Arquivo: diagrama_modelo.md
   - Descricao: Documentacao com a modelagem conceitual do grafo em sintaxe Mermaid e orientacoes para exportar a imagem oficial a partir do Neo4j Browser.

3. Backup e Exportacao de Dados:
   - Arquivo: backup_exportado.cypher
   - Descricao: Exportacao completa e atualizada dos dados da instancia em nuvem AuraDB do projeto.
   - Manual de Backup (.dump): tutorial_backup.md (contem o passo a passo para gerar o arquivo fisico .dump a partir do Neo4j Desktop local).
